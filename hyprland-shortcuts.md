## System Controls

| Shortcut | Action |
|----------|--------|
| `Super + Space` | Open app launcher (wofi) |
| `Super + Shift + Space` | Refresh waybar |
| `Super + W` | Kill active window |
| `Super + Escape` | Lock screen (hyprlock) |
| `Super + Shift + Escape` | Exit Hyprland |
| `Super + Ctrl + Escape` | Reboot system |
| `Super + Shift + Ctrl + Escape` | Power off system |

## Window Management

| Shortcut | Action |
|----------|--------|
| `Super + J` | Toggle split direction |
| `Super + P` | Toggle pseudo-tiling |
| `Super + V` | Toggle floating mode |
| `Super + ←/→/↑/↓` | Move focus between windows |
| `Super + Shift + ←/→/↑/↓` | Swap windows |

| `Super + =` | Resize window (width +100px) |
| `Super + Shift + -` | Resize window (height -100px) |

## Special Features

- **Special Workspace**: `Super + S` toggles a scratchpad-like workspace called "magic" where you can hide/show windows quickly
- **App Launcher**: `Super + Space` opens wofi for launching applications
- **Clipboard Manager**: `Ctrl + Super + V` opens clipse for clipboard history
